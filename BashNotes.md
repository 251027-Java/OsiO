#Bash Notes
pwd = print working directory
mkdir = create directory
cd = change directory
rmdir = remove directory
touch = create a file
ls = list directory contents
ls -a = list all including hidden
ls -l = list in a table or a list format
rm = remove/delete a file
rm -r = remove recursively 
rm -f = force (no no)
nano = open editor
vim = open the vim editor

## File Paths
~ = represents home directory
absolute path = file path starting with / that specifies a unique location
relative path = file path starting with . that defines the relation to the current working directory
.. = used with relative path to denote the parent directory (./..)
